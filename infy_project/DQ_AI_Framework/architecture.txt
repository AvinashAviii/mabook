## Architecture Overview

```
┌─────────────────────────────────────────────────────────────────────┐
│                        FastAPI Gateway                               │
│  /upload  /analyze  /validate  /report  /rules  /health             │
└──────┬──────────┬──────────┬──────────┬──────────┬──────────────────┘
       │          │          │          │          │
       ▼          ▼          ▼          ▼          ▼
┌──────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────────────┐
│  Data    │ │ Gemini  │ │  Rule   │ │ Report  │ │  Alert Engine   │
│  Loader  │ │ AI      │ │ Engine  │ │ Builder │ │ (Background)    │
│ (PySpark)│ │ Suggest │ │(PySpark)│ │ (JSON)  │ │  SNS/Email      │
└──────────┘ └─────────┘ └─────────┘ └─────────┘ └─────────────────┘
       │          │          │          │          │
       ▼          ▼          ▼          ▼          ▼
┌─────────────────────────────────────────────────────────────────────┐
│                     Shared State / Storage                           │
│   datasets{}   rules{}   results{}   anomalies{}   audit_trail[]   │
└─────────────────────────────────────────────────────────────────────┘
```

## Complete Project Structure

```
dq_framework/
├── main.py                    # FastAPI application entry point

├── requirements.txt           # Dependencies
├── docker-compose.yml         # Container orchestration
├── Dockerfile                 # Container image
│
├── core/
│   ├── __init__.py

│   ├── data_loader.py         # Data ingestion module
│   └── exceptions.py          # Custom exceptions
│
├── ai/
│   ├── __init__.py
│   ├── gemini_client.py       # Gemini API integration
│   ├── rule_suggester.py      # AI-powered rule suggestion engine
│   └── prompts.py             # Prompt templates
│
├── engine/
│   ├── __init__.py
│   ├── rule_parser.py         # JSON rule parser
│   ├── rule_executor.py       # PySpark-based rule execution
│   ├── validators.py          # Individual validation functions
│   └── conditional_logic.py   # Complex business logic handler
│
├── reporting/
│   ├── __init__.py

├── models/

├── tests/
```
